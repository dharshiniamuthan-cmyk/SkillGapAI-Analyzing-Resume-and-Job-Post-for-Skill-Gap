This mini project contains 8 individual Python programs that perform different resume-related text processing tasks.
Task List:

 Task 1 – Basic Skill Matcher
Compares resume skills with job-required skills and shows matched/missing skills.
 Task 2 – Resume Word Counter
Counts total words, unique words, and most repeated word.
 Task 3 – Experience Extractor
Extracts the number of years of experience from a sentence.
 Task 4 – Job Title Normalizer
Removes special characters and formats job titles properly.
 Task 5 – Resume Category Classifier
Classifies resumes into:
Data/ML
Web Development
Software Development
Other
 Task 6 – Skill Frequency Checker
Counts how many times a skill appears in the resume paragraph.
 Task 7 – Email Extractor (No Regex)
Finds email IDs by detecting “@” symbol.
 Task 8 – Tech Stack Counter
Counts how many technologies belong to languages, databases, or frameworks.
Each task is stored in a separate folder with its own Python file.
